Exercice 2 (b): Create BPMN model within the studio

This exercice is to add all the BPMN element to Bonita studio.

BPMN to process model in the studio

In this exercice, you need to create the BPMN model bellow in the studio.


Pool and Lane

  1. Double click on your process model to start the modeling of the BPMN model.

  2. Change pool name by click on the pool, then, within "General" tab, click on "pool" and then on the edit icon of the name of the lane.


  3. Add new lane to the process model.

  4. change the name of the lane by clicking on "General" tab, then "Lane" and then on the edit icon and change the lane name. Change the name of the lanes based on the process model above.

  5. Result:

Add Process Model Elements

After adding the pool and the lane, we need to add other elemenets (activities, gateways, and events).
  1. Change activity name and properties, by clicking on it, then within "General" tab, you can change the name of the activity, and the type "Task Type".

  2. Add next element by clicking on it, then select what element you want to add (activities, gateways, and events,...).

  3. You can also change the activity type by clicking on the activity and the on the setting icon, and select the type you need.

  4. You can also change the gateways type by clicking on the gateway and the on the setting icon, and select the type you need.

  5. Add all the elements of the process model to go to the next exercice and start the implementation of our process model.
