Lab for Business Process Management

This lab is to use Bonita soft for business process management, also to using DF-BPMN tool for modeling a process model with no hading code.

  • Bonitasoft tutorial
    1. Exercice 1: Bonitasoft installation
    2. Exercice 2: Modeling using Bonita studio
      • (a) Add the Business Data Mode (BDM-> UML)
      • (b) Create the process model (BPMN)
      • (c) Add the data variables
      • (d) Implementation
      • (e) Execution
    3. Exercice 3: Export Bonita project

  • DF-BPMN tutorial
    1. Exercice 1: DF-BPMN tool installation
    2. Exercice 2: Modeling using DF-BPMN Tool
      • (a) Add the Business Data Mode (BDM-> UML)
      • (b) Create the process model (BPMN)
      • (c) Convert To Bonita studio
      • (d) Execution
    3. Exercice 3: Export DF-BPMN Project
