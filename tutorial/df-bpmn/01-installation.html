Exercice 1: DF-BPMN tool installation

This exercice to install the DF-BPMN tool, test, and create the first project within this tool.

Before you start modeling, you must first install the DF-BPMN tool and Bonita studio. You should follow the instructions of Bonita Studio here. Then, you can start to follow the instraction bellow to install DF-BPMN tool.

You can build and install all of the environments manually or use the Docker version for a quick and easy system.

Here, we will illustrate how to install the DF-BPMN tool using Docker image:
  1. Download docker application and create account.
  2. Go to your terminal/cmd then run: docker pull alinoureldin/df-bpmn:latest
  3. Before run the next command you should:
    • Change <openai-key> in the next command (you was received by email).
    • Change the <bonita-wokspace> to the path of your bonita workspace folder.
  4. Run the following command after change the <openai-key> and <bonita-wokspace>: docker run --name="df-bpmn" --rm -p 3000:3000 -p 3001:3001 --env OPENAI_KEY=<openai-key> -it -v <bonita-wokspace>:/usr/src/app/bonita alinoureldin/df-bpmn


Testing for your tool

You can access the tool using the following link http://localhost:3000/#/usr/src/app/workspace

The image bellow is the default page for the tool.


Create New BPMN file

  1. Right click on the white space, and click on "New file"

  2. rename to *.bpmn


Double click on the created file, and now you can start modeling in DF-BPMN tool.
