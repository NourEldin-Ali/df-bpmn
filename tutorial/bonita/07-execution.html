Exercice 2 (e): Execution of the process model

This exercice is to execute the process model.

This is the last part to execute the process model, in this example we does not create groups for requestors and supplier, we use the same actor and same users within all the process.

Define lane actors

Click on the lane, then within "General" tab, click on "Actors" and select the existing actor.


Actors configuration

Click on the "Configuration", click on "Groups", select "/acme" to get all the users groups, then click on "finish" and click on "Finish" to save the configuration.





Now you are ready to execute the process model.

Process execution

  1. To execute the process model, click on "Run".
  2. Your browser will be automatically opened, then start a new process an then click on "Tasks".
  3. Select a task, and click on "Take"
  4. Fill the form, and click on "submit".
  5. Refresh the task, select the next task and continue the process until it finishes.
  6. You can validate the Done Tasks to make sure that all is good.
  7. Or you can open the DBM and validate the data stored within it.

    Then select a quotation or request to comfirm that the data are stored within the DBM.
